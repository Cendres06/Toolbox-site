Keybert

Cet outil utilise le modèle KeyBERT. Le texte est encodé au niveau de la phrase. Le modèle utilisé pour les transformers de phrases dans l'outil ci-dessous est distiluse-base-multilingual-cased-v1, qui prend en charge 15 langues. Pour diversifier les mots-clés, ils est possible d'appliquer deux algorithmes :

Maximal marginal relevance (MMR) permet de filtrer les mots-clés candidats selon leur taux de diversité. Un taux de diversité élevé évitera les redondances ; un taux de diversité faible pourra par exemple permettre de cibler des nuances autour de thèmes principaux.

Maximal sum similarity (MSS) sélectionne les mots-clés qui ont le taux de similarité deux à deux le plus faible possible.

Cet outil extrait les mots les plus utilisés dans un document ou dans un corpus. L’extraction des mots clés est un processus important dans les humanités numériques car elle permet d'identifier le contenu des textes qui peut être utile dans la constitution d’un corpus. Les mots clés extraits peuvent être aussi inclus dans les métadonnées du texte. D’autres raisons à utiliser cet outil incluent les options différentes possibles pour la modélisation des données. Les résultats peuvent être téléchargés en format .csv.
