Tanagra

Cet outil est destiné aux chercheurs qui voudraient visualiser des lieux géographiques qui se trouvent dans leur corpus. Tanagra extrait les entités nommées géographiques d’un corpus entier ou d’un seul fichier et les cartographie. Cet outil permet la sauvegarde de cette visualisation en format .png. L'utilisateur peut choisir entre une sélection des fonds cartographiques, par exemple un fond avec le terrain physique peut être utile pour un projet qui examine l'évolution des noms utilisés pour décrire l'environnement pour une intervalle des années. Également, les données géographiques extraites du corpus ou du texte peuvent être téléchargées en format .csv.

Tanagra sur Pandore

À partir d’un corpus de textes, l’application permet d’identifier les noms de lieu à l’aide de l'application SpaCy. Ensuite, les lieux reconnus sont géolocalisés en utilisant une base de données, puis projetés sur une carte interactive. Les nouveaux lieux peuvent également être ajoutés à la base de données. Il est possible pour l’utilisateur de filtrer les résultats par métadonnées (titre ou date), et d’exporter les lieux cartographiés au format csv.

1. Sélectionnez le modèle de SpaCy

2. Ajoutez votre fichier
La visualisation de sortie comprend plusieurs informations importantes. Les lieux géographiques sont affichés en ordre alphabétique. Si plusieurs options sont trouvées, vous pouvez sélectionner les plus pertinents. Par exemple entre Franche-Comté - Franche-Comté - France et Franche-Comté - Bourgogne-Franche-Comté - France.
